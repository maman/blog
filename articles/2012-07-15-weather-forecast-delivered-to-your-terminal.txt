---
title: Weather forecast, delivered to your Terminal
date: 15/07/2012

`finger` - [Manpage](http://man.cx/finger) - can be used to lookup the current weather forecast for your city by running  

    finger cityname@graph.no

for further info, run

    finger @graph.no

and to get one-liner weather report suitable to use in tmux statusline / screen hardstatus just run

    finger o:cityname@graph.no

###Info

for archlinux users, install package `netkit-bsd-finger` to get `finger` command ready.
